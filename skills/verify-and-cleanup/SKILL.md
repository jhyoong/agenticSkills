# Skill for final review of plan and completion
1. Loads the plan file
2. Break down key features to test and review
3. Launch subagent to check
4. If there are issues, send proposed fix and launch developer subagent to fix. Then, ask human to run this skill again in fresh context. 
5. If no issues, propose merge and cleanup to be done. Actual merge and delete will be done by human.
